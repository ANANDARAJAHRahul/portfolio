## 2. Research Projects

**Note:** The French "Grandes Écoles" system includes long-term research projects (6 months) equivalent in depth to a master's thesis.

### Optimal Transport *(Master 1 Project)*

📄 [View Report](./Projects/Optimal_Transport.pdf)

This project explores the theoretical foundations of optimal transport, including Wasserstein distances and Sinkhorn regularization. Applications include the comparison of complex probability distributions and potential use in **biomedical data integration and generative modeling**.

---

### Aitchison Algebra *(Master 2 Project)*

📄 [View Report](./Projects/Aitchison.pdf)

This project investigates compositional data analysis using Aitchison geometry. Such methods are particularly relevant for **omics data (e.g., microbiome, gene expression)**, where relative abundances require non-Euclidean representations.
